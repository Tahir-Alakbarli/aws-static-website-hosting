Small project, real cloud concepts.

I created this project to demonstrate static web hosting, content delivery, DNS configuration, TLS certificates, and secure access between AWS services.

DESIGN CHOICES

What this deployment demonstrates

Private origin

S3 Block Public Access remains enabled. CloudFront reads objects through Origin Access Control.

Global delivery

CloudFront caches content at edge locations, reducing repeated requests to the origin.

Custom HTTPS domain

Route 53 handles DNS while AWS Certificate Manager provides the TLS certificate.
